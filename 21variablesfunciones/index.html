Algúnas funciones en CSS

  • calc(): Realiza cálculos matemáticos.
  • rgb() / rgba(): Define colores usando valores de rojo, verde y azul (y opacidad con a).
  • hsl() / hsla(): Define colores usando matiz, saturación y luminosidad.
  • clamp(): Establece un rango de valores, con un mínimo, un preferido y un máximo.
  • url(): Define una URL para imágenes, videos o fuentes.
  • min() / max() Selecciona el valor mínimo o máximo de entre varios valores.
