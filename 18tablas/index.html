Propiedades de estilo comunes para tablas

Documentación oficial
  • border: Para bordes.
  • border-spacing: Separación entre las celdas (si no usas border-collapse: collapse)
  • padding: Espaciado dentro de las celdas.
  • text-align: Alineación del texto.
  • width y height: Ajustar el tamaño de la tabla o celdas.
  • background-color: Colores de fondo.
  • border-collapse: collapse;: Combina los bordes de las celdas para que se vean como una línea continua.
  • nth-child(even): Aplica estilos a las filas pares, creando un efecto de "zebra".
